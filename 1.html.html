HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browseres receive HTML documents from a web server or from local storgage and render them into multmedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create structured documnets by denoting structual semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elemets are delineated by tags, written using angle bracketss.
